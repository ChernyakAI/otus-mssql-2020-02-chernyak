Описание проекта.

База данных для системы учёта и расчётов за жилищно-коммунальные услуги (расчётный центр).
Основные сущности:
	адресные объекты;
	объекты технического учёта;
	потребители/физические лица;
	услуги и схемы расчёта;
	поставщики ресурсов.
Функциональность:
	ведение информации о жилищном фонде;
	ведение лицевых счетов;
	контроль межповерочного интервала приборов учёта и уведомление потребителей;
	расчёт начислений в соответствии с методиками расчёта услуги;
	приём оплат и распределение по поставщикам услуг;
	подготовка данных для печати платёжных документов;
	анализ дебиторской/кредиторской задолженности потребителей.
